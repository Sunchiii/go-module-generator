# Prepare Requirement

- golang
- MacOs OR Linux OR Windows

# Installation

```
git clone git@github.com:Sunchiii/go-module-generator.git
```

```
cd go-module-generator
```

### MacOs OR Linux

```
./install.sh
```

### Windows

```
install.bat
```

# Usege

> [!IMPORTANT]
> In your directory structure must has been "/src/"

```
go-gen <yourServiceName>
```
